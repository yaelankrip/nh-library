# ספריית נס הרים — הוראות העלאה לוורסל

## שלב 1 — התקנת נוד.ג'יי-אס (Node.js)
1. היכנסי לאתר: https://nodejs.org
2. לחצי על הכפתור הירוק הגדול (LTS)
3. הורידי והתקיני — לחצי "Next" בכל השלבים

## שלב 2 — פתיחת הפרויקט
1. חלצי את תיקיית `nh-library` למחשב שלך
2. פתחי את תוכנת הטרמינל (Terminal):
   - **מק**: חפשי "Terminal" בספוטלייט (Cmd+Space)
   - **וינדוז**: לחצי Win+R, כתבי `cmd`, אנטר
3. נווטי לתיקייה — כתבי:
   ```
   cd Desktop/nh-library
   ```
   (או הנתיב שבו שמרת את התיקייה)

## שלב 3 — התקנת חבילות
בטרמינל, הריצי:
```
npm install
```
המתיני כמה דקות עד שזה מסיים.

## שלב 4 — העלאה לוורסל
1. היכנסי לאתר https://vercel.com וצרי חשבון חינמי
2. לחצי על "Add New Project"
3. לחצי על "Upload" (העלאה ישירה)
4. גרירי את תיקיית `nh-library` לתוך החלון
5. לחצי Deploy — וזהו! 🎉

## עדכון בעתיד
בכל פעם שתרצי לשנות משהו באפליקציה:
1. ערכי את הקובץ `src/App.jsx`
2. היכנסי לוורסל → הפרויקט שלך → לחצי "Redeploy" והעלי את התיקייה המעודכנת

## שאלות?
